✦ How to Train Your Ear

The ear is a muscle. It learns through repetition, not theory.

Start by memorizing A440 — hum it, sing it, find it in the world around you.

Then learn its neighbors. A# sits just above. G# just below.

Over time the distances between notes become familiar. This is called relative pitch.

Absolute pitch — knowing a note without reference — takes years. Relative pitch can be learned in weeks.

✦ Sharps vs Flats

C# and D♭ are the same pitch — two names for the same key on the piano. Which name you use depends on the key you're in. In this game we use sharps.
